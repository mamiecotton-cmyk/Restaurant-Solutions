# Wally's NW Soul

> Northwest Soul. Real Flavor.

A modern, high-converting restaurant ordering website for Wally's NW Soul — a soul food restaurant in the Pacific Northwest.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Stripe Checkout
- Vercel

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your Stripe keys.

## Deploy

Deploy instantly on [Vercel](https://vercel.com).
